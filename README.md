# rock-scissors-paper-game
This is my forth project.

# How it works
- Firstly, the program shows rules of the game.
- You have to choose only one in rock or scissors or paper at once.
- If you win, the program will show it and otherwise it shows that computer wins.
- If you want to end the game enter -1
- Finally, the program shows your marks, computer marks and difference of them.
